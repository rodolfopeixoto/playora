# Catalog rules

Catálogo Playora aceita SOMENTE:

- `homebrew_game` — homebrew com licença permissiva
- `demo_authorized` — demos autorizadas (não ROMs comerciais)
- `open_source_game` — código aberto (MIT, GPL, Apache, BSD, etc.)
- `public_domain` — DP comprovado
- `metadata_pack` — covers/gamelists para uso pessoal
- `theme` — themes ES
- `config_pack` — configs RetroArch/runners
- `patch_allowed` — patches IPS/BPS sob licença permitida

PROIBIDO:
- ROM comercial protegida
- Cópia de jogos retail
- BIOS proprietária

A licença deve estar em `CatalogItem.license`. Server **não** distribui
artefatos cuja licença não seja explícita e auditável.
